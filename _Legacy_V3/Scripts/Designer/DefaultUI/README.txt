ATTENTION - TEMPLATE DE RÉFÉRENCE (GOLDEN MASTER)

Ce dossier contient le squelette "v2.0" standard pour tous les scripts de l'application.

NE MODIFIEZ PAS CES FICHIERS DIRECTEMENT pour créer un nouveau script.

Procédure :
1. COPIEZ ce dossier entier.
2. COLLEZ-le dans la catégorie de destination (ex: Scripts/UserManagement/).
3. RENOMMEZ le dossier, le fichier .ps1 et le fichier .xaml.
4. Modifiez le manifest.json (Nouvel ID unique obligatoire).

Référez-vous à "Docs/DEVELOPER_GUIDE.md" pour la procédure complète.
